# kt cloud TECH UP - 합격자 미션 가이드 웹사이트

## 프로젝트 개요
kt cloud TECH UP 최종 합격자를 위한 수강 전 필수 미션 가이드 웹사이트입니다. 9개 과정별로 맞춤형 가이드를 제공하며, 4가지 미션을 단계별로 안내합니다. 모바일 최적화 및 다양한 애니메이션 효과가 적용되어 있습니다.

## 주요 페이지

### 1. 시작 페이지 (`index.html`)
- 수강생 이름 입력
- 9개 과정 중 합격 과정 선택
- 과정 소개 카드
- localStorage 기반 정보 저장

### 2. 미션 가이드 페이지 (`mission.html`)
- 개인화된 환영 메시지
- 과정별 맞춤 HRD-Net 링크
- 4가지 미션 가이드
- 진행률 추적

## 지원 과정 (9개)

1. **프로덕트 디자인** 🎨 - UX/UI 디자인 전문가 양성
2. **프로덕트 매니지먼트** 📊 - 제품 기획 및 관리 전문가 양성
3. **클라우드 네이티브** ☁️ - 클라우드 네이티브 개발 전문가 양성
4. **클라우드 인프라** 🖥️ - 클라우드 인프라 구축 전문가 양성
5. **사이버 보안** 🔒 - 정보보안 전문가 양성
6. **생성형 AI** 🤖 - AI 서비스 개발 전문가 양성
7. **프론트엔드** 💻 - 프론트엔드 개발 전문가 양성
8. **풀스택** ⚡ - 풀스택 개발 전문가 양성
9. **백엔드** 🔧 - 백엔드 개발 전문가 양성

## 주요 기능

### ✅ 완료된 기능

1. **2단계 페이지 구조**
   - 시작 페이지: 이름 및 과정 선택
   - 미션 페이지: 개인화된 미션 가이드

2. **9개 과정별 맞춤 정보**
   - 과정별 HRD-Net 직접 링크
   - 과정명 표시
   - 개강일자 표시

3. **반응형 디자인**
   - 모바일, 태블릿, 데스크톱 모든 환경 지원
   - 유연한 레이아웃 및 터치 최적화

4. **4가지 미션 가이드**
   - Mission 1: 국민내일배움카드 & 고용24 수강신청
   - Mission 2: 디스코드 가입하기
   - Mission 3: ZEP 가입 및 입장하기
   - Mission 4: 구름EDU 가입 및 과정별 강좌 입장하기

5. **인터랙티브 기능**
   - 미션 완료 버튼 (토글 방식)
   - 진행률 표시 바
   - localStorage 기반 진행 상황 및 개인정보 저장

6. **애니메이션 효과**
   - Hero 섹션 별빛 배경 애니메이션
   - 로고 Glow 효과
   - 스크롤 기반 카드 등장 애니메이션
   - 미션 완료 시 Confetti 효과
   - 호버 시 카드 확대 및 이미지 줌
   - 버튼 펄스 애니메이션

7. **사용자 경험 개선**
   - 개인화된 환영 메시지
   - 과정별 맞춤 링크 제공
   - 단계별 체크리스트
   - 상세한 How-to 가이드
   - 스크롤 인디케이터
   - 부드러운 스크롤 애니메이션

## 파일 구조
```
├── index.html          # 시작 페이지 (이름/과정 선택)
├── mission.html        # 미션 가이드 페이지
├── css/
│   ├── start.css      # 시작 페이지 스타일
│   └── style.css      # 미션 페이지 스타일 (반응형, 애니메이션)
├── js/
│   ├── start.js       # 시작 페이지 기능 (과정 데이터, 폼 처리)
│   └── main.js        # 미션 페이지 기능 (진행률, 개인화, 애니메이션)
└── README.md          # 프로젝트 문서
```

## 기술 스택
- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 그라데이션, 애니메이션, 미디어 쿼리
- **Vanilla JavaScript**: 인터랙션, localStorage, Intersection Observer, 동적 콘텐츠
- **Font Awesome**: 아이콘
- **Google Fonts**: Noto Sans KR

## 사용자 플로우

1. **시작 페이지 접속** (`index.html`)
   - 이름 입력
   - 9개 과정 중 선택
   - "미션 시작하기" 클릭

2. **미션 페이지 이동** (`mission.html`)
   - 개인화된 환영 메시지 표시
   - 선택한 과정명 표시
   - 과정별 맞춤 HRD-Net 링크 제공

3. **미션 수행**
   - 4개 미션 순서대로 완료
   - 각 미션 완료 버튼 클릭
   - 진행률 자동 업데이트

4. **완료**
   - 모든 미션 완료 시 축하 섹션 표시
   - Confetti 애니메이션 실행
